# Absolute Cinema Reviews

Веб-сайт для публикации и модерации рецензий на фильмы  
*Курсовая работа по дисциплине «Теория разработки качественного программного обеспечения»*  
Санкт-Петербургский политехнический университет Петра Великого, 2025

---

## Описание

Пользователи могут:
- Просматривать одобренные рецензии (без авторизации)
- Регистрироваться и входить в систему
- Создавать, редактировать и удалять свои рецензии
- Ставить лайки одобренным рецензиям

Администраторы могут:
- То же, что и пользователи
- Модерировать рецензии (approved / rejected)
